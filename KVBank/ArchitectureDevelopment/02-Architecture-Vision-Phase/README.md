# Architecture Development - Phase A - Architecture Vision

## Steps for Phase A
1. Establish Architecture Project
2. Identify stakeholders, concerns, and requirements
3. Confirm Business Goals, Business Drivers, and Business Constraint
4. Evaulate Capabilities
5. Assess readiness for transformation
6. Define Scope
7. Confirm architecture principles, including business principels
8. Develop Architecture Vision
9. Define architecture value and KPIs
10. Identiy transformation risks and mitigation activities
11. Develop statement of Architecture work, secure approval

## Outputs from Phase A
1. Approved Statement of Architecture work
2. Refined statements of Business Principles, Goals and Drivers
3. Architecture Principles
4. Capability Statement
5. Tailored Architecture Framework

## Artifacts Produced
1. Matrices: Stakehoders Matrix
2. Diagrams: Business model diagram, Business Capability Map, Value Stream Map, Value Chain Diagram, Solution Concept Diagram
